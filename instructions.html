Gameplay

Classic pong rules apply.

The game is continuous and doesn't stop.

There is no official point limit. Play as long as you want.

Gameplay is really simple. Object descriptions are below if anyone desires detail.

Controls

Player 1 (Top Paddle):

  • Move Left/Right: A/D keys
  • Shoot gun: W key
  • Launch ball: S key

Player 2 (Bottom Paddle):

  • Move Left/Right: Left/Right arrow keys
  • Shoot gun: Up arrow key
  • Launch ball: Down key

Game Objects

Ball:

  • Each player can launch a ball every 3 seconds.
  • Balls will bounce off each other.

Gun:

  • Each player can shoot his/her gun every 3 seconds.
  • Gun bullets can:
  • Destory balls.
  • Stun another player's paddle (for 1.5 seconds).
  • Temporarily destroy another player's gun(s).
  • A player's destroyed gun will regenerate upon his/her next point.

Version 2 Update

Bug Fixes/Polishes:

  • Balls will stick together significantly less often.
  • Points are not scored until the balls fully cross the canvas's edge.

Visuals:

  • Electricity buzzes around a paddle when it's stunned.
  • The velocity for bullets has been increased. Having the ball and bullet velocities similar sometimes forced a player to get stunned while reflecting a ball, or avoid getting stunned but consequently miss the ball.

Sounds:

  • Launching a ball
  • Shooting a gun
  • Breaking a ball
  • Breaking a gun
  • Stunning an opponent
  • Scoring a point

Known bugs

Balls will occasionally stick together when they collide.

Balls will occasionally go through a paddle if the balls are very close together.

If a player's gun is destroyed and the paddle is stunned by the same bullet, the sound for breaking the gun will not play.
